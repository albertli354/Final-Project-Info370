Training a simple MLP Classifier¶

Before investing a lot of time in training a more complicated Neural Network, we'll start with a simple Scikit Learn MLP Classifier. We'll begin by loading one of the CSVs generated from the MNIST Special Database 19 original images and matched to provide writer, character and character class labels.

Some of these columns data types are not inferred well by pandas, so we'll load them all as strings, and then explicitly set the types.

There are 10 authors in the loaded dataset with 3812 samples and a min/max of 315/418 samples per author.

Next, the dataset will be split into training (75%) and test (25%) sets so we can evaluate perfomance rather than overfitting to the data.

Now we'll train the classifier with a 3 hidden layer perceptron net (256, 256, 128), using the default "adam" solver, and a max iteration of 20 for times sake on each of these randomized label datasets to determine what predictions we would see on an individual writer's writing samples, this will give us a distribution in the $H_0$ case that "everyone writes the same".
