Reflection

Through the project, we gained valuable experience in machine learning. We have several things that would like to share with you. First of all, we learned that machine learning is an iterative process and it is really important to build a simple model in the first and then improve it iteratively. Since we are not the experts in the field of image recognition, it was hard for us to find the right direction. In this case, we first build a model and use it to find the clues to improve our model.

Another important thing is to keep the evaluation standard consistent as much as possible. During the modelling, we used different approaches to calculate the accuracy. However, since the standards are different, it was really hard for us to directly compare performance for different models. We learned that using a simple method to evaluate the results is actually the best approach.

In the future, we may build up our own model to test the accuracy and facilitate these industries which authenticate and identify the users who use their system or product.
