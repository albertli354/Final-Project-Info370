We used heatmap to build correlation matrix. It shows that each pixel column that we selected has similar correlation to the writer id. Therefore, we will combine these pixel columns to build a statistical model.

We used multivariable linear regression to model the relationship between pixels of images and the writer. The independent variables are pixel columns that were selected using Lasso, and the dependent variable is the writer id. Below is the summary of the model.

We can see that the R-squared is 26%, which means the model only explains 26% variability of the response variable. Low R-squared value indicates that the model does not fit the data well. By looking at the p-value for each independent variable, we can see that pixel_5319, pixel_5448, pixel_7503, pixel_7730, pixel_7858, pixel_8013, pixel_10174 and pixel_10419 are not statistically significant since their p-value are greater than alpha level 0.05, which means the model does not gain enough information from having these variables.
