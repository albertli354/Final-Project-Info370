Data Source

The dataset we are using is NIST Special Database 19. It consists of 5 zipped files with a total of 3,992,357 images. The dataset completed by Bethesda High school students and Census Bureau employees in Suitland, Maryland. Every participant asked to write all alphanumeric characters in a case-sensitive manner. We will use the dataset classified by author (the by_write organization of the dataset per the description in https://s3.amazonaws.com/nist-srd/SD19/sd19_users_guide_edition_2.pdf), meaning we have the labels that tell us the image belong to different authors. However, all personal identifying information was removed in the dataset.
